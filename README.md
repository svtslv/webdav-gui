## Installation

```bash
git clone https://github.com/svtslv/webdav-gui
```

```bash
cd webdav-gui
```

```bash
npm install
```

### MacOS

```bash
npm run electron:mac
```

### Windows

```bash
npm run electron:windows
```

### Linux

```bash
npm run electron:linux
```
